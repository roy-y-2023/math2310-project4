# trips-recorder

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.37. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

# Operations

## Get Trips
```bash
bun run get-trips.ts
```

## Get Status of Trips
```bash
bun run get-trips-status.ts
```
